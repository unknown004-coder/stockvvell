# Stockvvell

Stockvvell is a simple inventory management system developed as a college-level project.
It helps users manage products, track stock status, and view basic sales information.

## Features
- Login page
- Dashboard with summary cards
- Inventory management
- Sales & demand overview
- Alerts section

## Tech Stack
- HTML
- CSS
- JavaScript

## Note
This project is in an early stage and all data starts from zero.
